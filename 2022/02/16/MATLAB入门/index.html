MATLAB入门

MATLAB入门

基本常识

MATLAB默认数据类型是double

Inf：无限大 eps：2.2204e-016无限小

不加“；”，则会打印出该等式的结果值

Matlab索引从 1 开始

Calling Priority

%% ：区分一个section

一行打不完 用… 可以在第二行继续写

数组

条件表达式

注意都要有end结尾

  • if elseif else
  • switch case otherwise
  • while
  • for n = a:increment:c

函数

function [A B] = …表示结果是多个结果

function handle

f = @(x) exp(-2*x);
x=0:0.1:2;

相当于是函数指针 f是指向函数exp(-2*x)的指针

矩阵

矩阵查询

A=[1,2,3;4,5,6;6,7,8],对于通过A(x)查找矩阵中的某个数，是按照列从上至下的顺序来的，而非行
A([a b c])可查找多个数
A([a b;a c;])得到矩阵
A([a b],[a c]) row a b 和 col a c的交集

矩阵变换

替换某个数：A(r,c)=x
等差数列应用 A=[1:5; 2:3:15;-2:0.5:0]

对A矩阵，A(3,:)=[]可将该行抹掉，达到删除的效果

矩阵运算

A.*B：直乘，结果矩阵的（i , j）=A(i,j) * B(i,j)
./ .^：效果同上

便捷指令

  • eye(n)：n维单位矩阵
  • zeros(r,c)：零矩阵
  • ones(r,c)：全为1的矩阵
  • diag([a,b,c])：对角矩阵
  • max(A)：找到每个col中最大的数
  • sum min mean（求平均）同上
  • sort(A)：不改变A，另外创建一个矩阵 对每个col按升序排序
  • sortrows(A)：基于第一列的元素按升序给矩阵排序，注意这里是一整行的移动，与sort不同
  • size(A)：求row 和 col数
  • length(A)：求row，即vector
  • find(A==x)：求矩阵A中等于x的位置（索引方式返回）

运算符

*一定要注意./ .的应用，考虑引入向量是否会出错

‘：’ (Colon)

等差数列 A=[1:100] first = 1 end = 100 step = 1
A=[1:2:100] first = 1 end = 100 step = 2
与字符混用 str=‘a’:2:‘z’

表全部：A(3,:)表示矩阵A的row 3的全部

~=： 等于！=

变量

变量类型

Matlab类型转换 通过函数进行 例如int16(x)

字符串

字符串连接Concat: [s1,s2]

字符串中字符的比较 str=='a’是str每个字符与’a’进行比较，且str==‘a’与’a’==str有不同的意义，一个是一维的比较一个是多维的比较

字符串查找替换：str(str==‘l’) = ‘L’
str=='l’返回的是逻辑结果，例如11000010，指示了对应的索引位置—多索引位置的指示

结构体

可用struct函数去声明一个struct（‘name’,value,‘name’,value,…），也可以直接Student.name = …声明

结构体复用：Student(2).name …
这时，Student就从单个变量，变成一维向量了

结构体函数

  • rmfield
  • setfield
  • struct
  • sturct2cell
  • fieldnames
  • getfield
  • isfield
  • …

Cell Array

声明一个Cell Array

A(1,1)={xxxx};
A(1,2)={xxxx};
A(2,1)={xxxx};
A(2,2)={xxxxx};
% 也可以用这种方式A{1,1}=xxxx;

每个Entry都有一个pointer指向对应的数据

查看Cell Array中的数据类型：A(1,1)
查看Cell Array中的数据值：A{1,1}

Function

  • cell：创建一个Cell Array
  • cell2mat
  • cell2struct
  • cekkduso
  • cellfun
  • cellplot
  • cellstr
  • num2cell

Multidimension Array

三维及以上的结构体，除了row 和 col外还有layer等

A{1,1,1}=xxx;声明一个

或者用cat连接：

cat(x,A,B)：A B为要连接的Array
x的值代表不同的维度：1->row;2->col;3->layer

重塑reshape(A,r,c)：将A Array变换成不同row col排布的Array

进阶

File Access

  • save mydata.mat -ascii ：将workspace所有data存入目标文件，提供不同编码格式的存入
  • load(‘filename’,‘-ascii’)：加载mat数据
    • 还可以读取xlsx文件的指定行列，比较牛逼
  • xlswrite(‘filename.xlsx’,value,sheetpage,‘E2:E4’)：写数据到xlsx
  • xlsread：读取xlsx文件 [Score Header]=xlsread(filename)
File I/O
  • fopen(filename,‘r’)：返回fid
  • fclose
  • fscanf(fid,type(‘%d’),size)
  • fprintf(fid,format,value)
  • feof(fid)：检测文件是否结束
    • while ~feof(fid)常用

Graphics

MATLAB不会直接通过表达式画图，而是通过点来画图

plot

plot(x,y, format) / plot(y) format详细信息搜索linspec

legend() 为曲线添加图例

title() 添加标题名称
xlabel() ylabel() zlabel()
text(0.25，2.5，str， ‘Interpreter’，‘latex’) 通过LaTex显示特殊的数学表达式 其中str=‘$ $xxxxx$ $’
annotation() 指示箭头

axis normal /equal(x y单位长度一致)/ square(x y 测量长度一样) /tight 限制坐标轴与图像的贴合程度

hold on hold off 使得能够在一个Figure里展现多个plot图
figure,plot(x,y) 使得画两个图，但gca和gcf就是新的了
figure(1) figure(2) 指定不同的figure
grid on off 打开网格

在一个figure上画几个figure：subplot(row,col,index);

Advanced

Logarithm Plots

semilogx semilogy loglog：以坐标轴的对数差距显示

x=logspace(-1,1,100);
y=x.^2;
subplot(2,2,1);
plot(x,y);
set(gca,'XTick',[0:1:10])
title('Plot');
subplot(2,2,2);
semilogx(x,y);
title('Semilogx');
subplot(2,2,3);
semilogy(x,YData=y);
title('Semilogy');
subplot(2,2,4);
loglog(x,YData=y);
title('LogLog');
set(gca,'XGrid','on')
A=get(gca)
